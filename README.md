This is a Data Analytics project I have done using SQL as part of Codebasics SQL Project Challenge.

 PROBLEM 
Atliq Hardware is a leading manufacturer of computer hardware in India and other nations, but it struggles to make prompt and wise choices based on data. The management intends to conducts a SQL challenge to identify applicants with both technical and soft skills.

 TOOLS & APPROACH 
MySQL, PowerBI, PowerPoint
路 MySQL - Concepts like Joins, CTEs, Subqueries, Window functions, String functions, Numeric functions, Date functions, etc.
路 PowerBI - Dax and visuals like Sunburst Chart, Bar Chart, Matrix, Line Chart, Maps, Donut Chart, etc.
路 PowerPoint- to present the Insights.

 FINDINGS & SOLUTIONS 
路 Findings - Important metrics such as Gross Profit, Contribution %, Overall Amount Sold, Average Gross Sales, Average Manufacturing Cost, etc.
路 Solutions - Increasing production rate by enticing customers with memberships, and other special offers.
